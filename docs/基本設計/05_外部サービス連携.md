## 1. Formspree（問い合わせフォーム）

| 項目 | 設定 |
|------|------|
| サービス | [Formspree](https://formspree.io/) |
| Form ID | `xwvnnqjj` |
| エンドポイント | `https://formspree.io/f/xwvnnqjj` |
| 使用ページ | `/contact/`, `/security/` |

### 1.1 スパム対策

| 対策 | 実装 |
|------|------|
| ハニーポット | `website` フィールド（`type="hidden"`、botが入力すると送信拒否） |
| レート制限 | 30秒間隔（フロントエンドで制御） |

### 1.2 エラーハンドリング

- 送信成功: トースト通知（上部中央、2.5秒で自動消失）
- 送信失敗: `sales@balcony.co.jp` への直接メールを案内

## 2. Google Analytics 4

| 項目 | 設定 |
|------|------|
| 測定ID | `G-55QTMHWXX0` |
| 読み込み | `<script async>` でGTMから配信 |


## 4. Cloudflare Workers（ヘッダーチェックAPI）

| 項目 | 設定 |
|------|------|
| エンドポイント | `https://header-check.balcony-workers.workers.dev` |
| 使用ページ | `/security/`（無料診断プレビュー）、`/tools/header-check/` |
| 用途 | 指定URLのHTTPセキュリティヘッダーを取得・分析 |

### 4.1 CSP 許可

セキュリティLP の CSP `connect-src` に追加が必要:

```
connect-src: https://header-check.balcony-workers.workers.dev
```
